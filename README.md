# @useless-javascript-modules/sum

This useless javascript module sums numbers.

## Example:	

```js
const sum = require('@useless-javascript-modules/sum')
sum([1,2,3,4]) /* = 10 */
sum([1.1,2.3,3.1,4.9]) /* = 11.4 */
```
